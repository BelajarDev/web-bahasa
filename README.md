# Jurusan Bahasa - Politeknik Negeri Bengkalis

[![POLBENG](https://img.shields.io/badge/POLBENG-Official-blue?style=flat&logo=polbeng&logoColor=white)](https://polbeng.ac.id)
[![Modern UI](https://img.shields.io/badge/Modern%20UI-Glassmorphism-brightgreen)](https://github.com/)
[![Responsive](https://img.shields.io/badge/Responsive-Mobile%20First-purple)](https://github.com/)

Website resmi **Jurusan Bahasa Politeknik Negeri Bengkalis** dengan design modern glassmorphism, animasi halus, dan konten lengkap.

## 🚀 Fitur Utama

- 🎨 **Design Modern**: Glassmorphism + AOS animations + Tilt effects
- 📱 **Fully Responsive**: Desktop, Tablet, Mobile
- ⚡ **Performance Optimized**: Clean code + CDN Bootstrap
- 🔗 **SEO Friendly**: Meta tags + Semantic HTML

## 📂 Struktur File

```
d:/WEBSITE_BAHASA/
├── index.html              (Beranda)
├── profil.html             (Profil Jurusan)
├── prospek-karir.html      (Prospek Karir D3/D4)
├── fasilitas.html          (15 Fasilitas + Lab)
├── dosen-tendik.html       (Dosen & Tendik)
├── akademik.html           (8 Layanan Akademik)
├── alumni.html             (Testimoni + Organisasi)
├── assets/
│   ├── css/style.css       (Custom theme: Blue/Emerald)
│   ├── js/script.js        (Interactions + Tilt + Magnetic)
│   ├── img/                (Assets - logo, photos)
```
